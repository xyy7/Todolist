# 移动端待办事项应用 (Todo App)

一个简洁、响应式的待办事项管理应用，专为移动端设计，支持任务增删改查、筛选、自动转移和导出功能。

## 功能特性

### 核心功能
- **任务管理**：添加、编辑、删除、标记完成/未完成
- **智能筛选**：按今天、本周、本月、全部视图筛选任务
- **日期规划**：为每个任务指定计划日期（支持日期选择器）
- **自动转移**：自动检测过去未完成任务，可一键转移到今天
- **本地存储**：使用浏览器 localStorage 持久化数据，无需后端
- **响应式设计**：适配移动端和桌面端，提供流畅的触摸体验

### 高级功能
- **设置面板**：可配置自动转移、默认视图、调试模式等
- **导出功能**：支持将任务数据导出为 JSON、CSV 文件或复制到剪贴板
- **调试面板**：内置实时日志查看器，便于开发和故障排查
- **鼓励动画**：完成当日所有任务时显示鼓励消息和烟花效果
- **时区处理**：所有日期操作均基于本地时区，避免 UTC 偏差

### 导出功能详解
- **JSON 导出**：完整任务数据（包括 ID、内容、计划时间、完成状态、完成时间），适合备份或迁移
- **CSV 导出**：表格格式，便于在 Excel、Numbers 等电子表格软件中打开和分析
- **剪贴板复制**：将任务摘要（含状态和日期）复制为纯文本，方便粘贴到其他应用

## 快速开始

直接打开 `index.html` 文件即可使用（需现代浏览器支持）。

### 本地运行
1. 克隆或下载项目文件到本地目录
2. 确保以下文件存在：
   - `index.html`

3. 用浏览器打开 `index.html`

## 使用方法

### 添加任务
1. 在输入框中输入任务内容
2. 选择计划日期（默认为今天）
3. 点击“添加”按钮或按回车键

### 管理任务
- **标记完成**：点击任务前的复选框
- **编辑内容**：直接点击任务文本进行编辑，失焦后自动保存
- **删除任务**：点击任务右侧的“×”按钮
- **筛选视图**：点击顶部筛选按钮（今天/本周/本月/全部）

### 导出任务
1. 点击右下角“设置”按钮打开设置面板
2. 在“导出任务数据”区域选择导出格式（JSON/CSV/剪贴板）
3. 点击“导出”按钮
   - 选择 JSON 或 CSV 将下载文件
   - 选择剪贴板将复制任务摘要，并显示提示

### 设置说明
- **转移过去未完成任务到今天**：启用后，每次加载页面时自动将过期未完成任务转移到今天
- **不展示转移提醒**：启用后，不再显示转移确认对话框，直接应用设置
- **启动默认视图**：设置应用启动时默认显示的筛选视图
- **调试模式**：显示调试面板，查看实时日志

## 数据模型

每个任务对象包含以下字段：

```javascript
{
  id: "1642166400000",        // 唯一标识（时间戳）
  content: "完成项目文档",     // 任务内容
  plannedTime: "2025-07-10",  // 计划日期（YYYY-MM-DD）
  completed: false,           // 完成状态
  completedTime: null         // 完成时间（ISO 字符串，仅当 completed 为 true 时）
}
```

## 导出功能实现细节

### JSON 导出 (`exportToJSON()`)
- 使用 `JSON.stringify(todos, null, 2)` 美化输出
- 通过 `Blob` 和 `URL.createObjectURL` 创建可下载文件
- 文件名包含日期：`todo-export-YYYY-MM-DD.json`

### CSV 导出 (`exportToCSV()`)
- 手动构建 CSV 字符串，包含表头和数据行
- 对内容中的引号进行转义（`"` → `""`）
- 使用 `text/csv` MIME 类型
- 文件名：`todo-export-YYYY-MM-DD.csv`

### 剪贴板复制 (`copyToClipboard()`)
- 使用 `navigator.clipboard.writeText()` API
- 生成易读的任务摘要（每行一个任务，含状态图标和日期）
- 提供视觉反馈（右上角 toast 提示）

## 常见问题

### 任务日期显示不正确？
- 应用使用本地时区处理所有日期，确保系统时区设置正确
- 日期格式为 YYYY-MM-DD，与 `<input type="date">` 兼容

### 导出文件无法下载？
- 检查浏览器是否阻止弹出窗口（允许下载）
- 确保有足够的磁盘空间

### 剪贴板复制不工作？
- 需要 HTTPS 环境或 localhost（某些浏览器限制）
- 可尝试手动复制控制台输出的内容
